## 템플릿

```markdown
# NNNN. (결정 제목)

- 상태: 제안 / 채택 / 폐기 / 대체됨(→ NNNN)
- 날짜: YYYY-MM-DD

## 맥락 (Context)
어떤 문제·상황에서 이 결정이 필요한가?

## 고려한 선택지 (Options)
- A: ... — 장점 / 단점
- B: ... — 장점 / 단점
- C: ... — 장점 / 단점

## 결정 (Decision)
무엇을 택했나.

## 근거 / 트레이드오프 (Rationale)
왜 그것을 택했나. 무엇을 얻고 무엇을 포기했나.

## 결과 (Consequences)
이 결정으로 생기는 후속 영향·제약.
```
